# Mobile & Touch Accessibility (14 checks)

**Standards**: WCAG 2.5.1 (A), 2.5.2 (A), 2.5.3 (A), 2.5.4 (A), 2.5.7 (AA), 2.5.8 (AA)

## 12.1 Touch Target Size (WCAG 2.5.5/2.5.8 - Level AAA/AA)

**Pass criteria:**
- [ ] Touch targets ≥ 24x24 pixels (AA)
- [ ] Touch targets ≥ 44x44 pixels (AAA)
- [ ] Adequate spacing between targets

## 12.2 Pointer Gestures (WCAG 2.5.1 - Level A)

**Pass criteria:**
- [ ] All path-based gestures have alternative
- [ ] Multi-point gestures have single-point alternative
- [ ] No swipe-only interactions

## 12.3 Pointer Cancellation (WCAG 2.5.2 - Level A)

**Pass criteria:**
- [ ] Actions completed on up-event (not down-event)
- [ ] Actions can be cancelled before completion
- [ ] Essential down-event actions identified

## 12.4 Label in Name (WCAG 2.5.3 - Level A)

**Pass criteria:**
- [ ] Visible labels included in accessible names
- [ ] Visible text matches programmatic label

## 12.5 Motion Actuation (WCAG 2.5.4 - Level A)

**Pass criteria:**
- [ ] Shake/tilt functionality has alternative
- [ ] Motion can be disabled

## 12.6 Dragging (WCAG 2.5.7 - Level AA)

**Pass criteria:**
- [ ] Drag operations have single-pointer alternative
- [ ] Click-based alternatives available
