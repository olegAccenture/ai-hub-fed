# Consistent Identification & Predictability (10 checks)

**Standards**: WCAG 3.2.1 (A), 3.2.2 (A), 3.2.3 (AA), 3.2.4 (AA), 3.2.6 (A)

## 13.1 On Focus (WCAG 3.2.1 - Level A)

**Pass criteria:**
- [ ] Receiving focus doesn't initiate unexpected change
- [ ] Focus doesn't trigger navigation
- [ ] Focus doesn't submit forms

## 13.2 On Input (WCAG 3.2.2 - Level A)

**Pass criteria:**
- [ ] Changing input doesn't cause unexpected action
- [ ] Form doesn't submit on selection
- [ ] Selection doesn't navigate away
- [ ] Warnings provided for automatic actions

## 13.3 Consistent Navigation (WCAG 3.2.3 - Level AA)

**Pass criteria:**
- [ ] Navigation consistent across pages
- [ ] Navigation in same relative order

## 13.4 Consistent Identification (WCAG 3.2.4/3.2.6 - Level AA/A)

**Pass criteria:**
- [ ] Same functionality labeled consistently
- [ ] Same icons have same meaning
- [ ] Same components identified consistently
