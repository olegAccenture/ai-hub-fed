# Links & Navigation (15 checks)

**Standards**: WCAG 1.4.1 (A), 2.4.4 (A), 2.4.5 (AA), 2.4.8 (AA), 2.4.9 (AAA), 3.2.3 (AA)

## 8.1 Link Purpose (WCAG 2.4.4/2.4.9 - Level A/AAA)

**Pass criteria:**
- [ ] All link text describes destination/purpose
- [ ] No generic link text ("click here", "more", "learn more")
- [ ] Link purpose clear from context
- [ ] Repeated generic links have contextual aria-label

## 8.2 Multiple Ways (WCAG 2.4.5 - Level AA)

**Pass criteria:**
- [ ] At least two navigation methods available
- [ ] Site search function works

## 8.3 Consistent Navigation (WCAG 3.2.3 - Level AA)

**Pass criteria:**
- [ ] Navigation consistent across pages
- [ ] Navigation in same relative order

## 8.4 Current Location (WCAG 2.4.8 - Level AA)

**Pass criteria:**
- [ ] Current page indicated in navigation
- [ ] Current page has aria-current="page"
- [ ] Breadcrumbs show current location

## 8.5 Link Appearance (WCAG 1.4.1 - Level A)

**Pass criteria:**
- [ ] Links distinguishable from regular text
- [ ] Link distinction not by color alone
- [ ] Link underlines or indicators present
- [ ] Underline on normal state (not just hover)
