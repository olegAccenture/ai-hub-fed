# Images & Alternative Text (18 checks)

**Standards**: WCAG 1.1.1 (A), 1.3.1 (A), 1.4.5 (AA), 1.4.9 (AAA)

## 5.1 Image Alternative Text (WCAG 1.1.1 - Level A)

**Pass criteria:**
- [ ] All images have alt attributes
- [ ] Alt text describes image purpose (not generic)
- [ ] Decorative images have empty alt (alt="")

## 5.2 Complex Images (WCAG 1.1.1 - Level A)

**Pass criteria:**
- [ ] Complex images have detailed descriptions
- [ ] Long descriptions accessible and complete
- [ ] Data visualizations have table alternatives

## 5.3 Images of Text (WCAG 1.4.5/1.4.9 - Level AA/AAA)

**Pass criteria:**
- [ ] Text used instead of images of text (where possible)
- [ ] Call-to-action buttons use actual text
- [ ] Logos and essential images of text identified

## 5.4 Background Images (WCAG 1.1.1 - Level A)

**Pass criteria:**
- [ ] Background images don't convey information
- [ ] Functional background images have text alternatives
- [ ] Social media icon backgrounds have alternatives

## 5.5 SVG Accessibility (WCAG 1.1.1 - Level A)

**Pass criteria:**
- [ ] SVGs have title/desc elements
- [ ] Interactive SVGs have proper roles and labels
- [ ] Clickable map SVGs properly labeled

## 5.6 Image Tag Misuse (WCAG 1.1.1/1.3.1 - Level A)

**Pass criteria:**
- [ ] <i> tags not used for images
- [ ] Informative images not CSS-only
- [ ] No informative content in CSS
