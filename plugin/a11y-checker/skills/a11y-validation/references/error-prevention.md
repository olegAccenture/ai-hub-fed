# Error Prevention & Recovery (12 checks)

**Standards**: WCAG 3.3.4 (AA), 3.3.6 (AAA), 3.3.7 (A)

## 14.1 Error Prevention (WCAG 3.3.4/3.3.6 - Level AA/AAA)

**Pass criteria:**
- [ ] Legal/financial submissions are reversible
- [ ] Important actions have confirmation dialogs
- [ ] Submissions can be checked before finalization
- [ ] Delete actions have confirmation
- [ ] Undo available for important actions

## 14.2 Redundant Entry (WCAG 3.3.7 - Level A)

**Pass criteria:**
- [ ] Previously entered information auto-filled
- [ ] Repeat information not required
- [ ] Form data persists across steps

## 14.3 Timeout Warnings (WCAG 2.2.1/2.2.3 - Level A/AAA)

**Pass criteria:**
- [ ] Session timeout warnings provided
- [ ] Time limits can be extended
- [ ] Users can request more time
- [ ] No time limits or adjustable to at least 10x
