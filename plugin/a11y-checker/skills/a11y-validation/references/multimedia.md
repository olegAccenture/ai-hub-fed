# Multimedia & Time-Based Media (13 checks)

**Standards**: WCAG 1.2.2 (A), 1.2.3 (A), 1.2.4 (AA), 1.2.5 (AA), 1.2.8 (AAA), 1.4.2 (A), 2.1.1 (A)

## 9.1 Audio/Video Controls (WCAG 2.1.1/1.4.2 - Level A)

**Pass criteria:**
- [ ] Media players have accessible controls
- [ ] Play/pause controls keyboard accessible
- [ ] Volume controls keyboard accessible

## 9.2 Captions (WCAG 1.2.2/1.2.4 - Level A/AA)

**Pass criteria:**
- [ ] Prerecorded video has synchronized captions
- [ ] Captions are accurate and complete
- [ ] Caption timing is synchronized
- [ ] Live video has live captions

## 9.3 Audio Descriptions (WCAG 1.2.3/1.2.5 - Level A/AA)

**Pass criteria:**
- [ ] Video has audio descriptions OR text alternative
- [ ] Visual information conveyed in audio/text

## 9.4 Transcripts (WCAG 1.2.8 - Level AAA)

**Pass criteria:**
- [ ] Transcripts available for all media
- [ ] Transcripts complete and accurate

## 9.5 Auto-play (WCAG 1.4.2 - Level A)

**Pass criteria:**
- [ ] No auto-playing audio > 3 seconds
- [ ] Auto-play can be paused/stopped
