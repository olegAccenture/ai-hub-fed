# Help & Documentation (11 checks)

**Standards**: WCAG 3.3.5 (AAA), 3.3.6 (AAA)

## 18.1 Help Access (WCAG 3.3.5 - Level AAA)

**Pass criteria:**
- [ ] Help available on every page
- [ ] Help consistently positioned
- [ ] Help clearly labeled
- [ ] Contact methods accessible

## 18.2 Context-Sensitive Help (WCAG 3.3.5 - Level AAA)

**Pass criteria:**
- [ ] Field-level help available
- [ ] Format instructions provided
- [ ] Examples provided for complex inputs
- [ ] Help icons/tooltips present

## 18.3 Documentation Accessibility (WCAG 1.1.1/1.3.1 - Level A)

**Pass criteria:**
- [ ] Documentation is accessible
- [ ] Instructions have alternatives
- [ ] Help content keyboard accessible
