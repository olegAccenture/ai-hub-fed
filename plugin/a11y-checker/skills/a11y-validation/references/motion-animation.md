# Motion & Animation (10 checks)

**Standards**: WCAG 2.2.2 (A), 2.3.1 (A), 2.3.3 (AAA)

## 11.1 Pause, Stop, Hide (WCAG 2.2.2 - Level A)

**Pass criteria:**
- [ ] Auto-updating content can be paused
- [ ] Moving content can be stopped
- [ ] Timing can be controlled by user

## 11.2 Animation from Interactions (WCAG 2.3.3 - Level AAA)

**Pass criteria:**
- [ ] Motion animations can be disabled
- [ ] prefers-reduced-motion media query respected

## 11.3 No Flashing (WCAG 2.3.1 - Level A)

**Pass criteria:**
- [ ] No content flashes more than 3x per second
- [ ] No general flash threshold violations
- [ ] No red flash threshold violations

## 11.4 Parallax & Effects (WCAG 2.3.3 - Level AAA)

**Pass criteria:**
- [ ] Parallax effects can be disabled
- [ ] Page usable with motion disabled
