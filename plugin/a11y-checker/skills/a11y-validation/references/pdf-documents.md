# PDF & Document Accessibility (15 checks)

**Standards**: WCAG 1.1.1 (A), 1.3.1 (A), 2.4.2 (A)

## 17.1 PDF Structure (WCAG 1.3.1/2.4.2 - Level A)

**Pass criteria:**
- [ ] PDF is tagged (not scanned image only)
- [ ] PDF has title
- [ ] PDF has language set
- [ ] Logical reading order maintained

## 17.2 PDF Content (WCAG 1.1.1/1.3.1 - Level A)

**Pass criteria:**
- [ ] PDF images have alt text
- [ ] PDF forms have labels
- [ ] PDF tables have headers
- [ ] PDF headings properly structured
- [ ] PDF links have descriptive text

## 17.3 PDF Alternatives (WCAG 1.1.1 - Level A)

**Pass criteria:**
- [ ] Complex PDFs have HTML alternatives
- [ ] PDF-only content is accessible
- [ ] Accessible versions clearly linked
